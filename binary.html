Binary Basics

The binary system uses only two digits: 0 and 1. Each digit is called a bit. A group of 4 bits is called a nibble; 8 bits make a byte.

Bits, Nibbles and Bytes

A bit is the smallest unit of data in digital systems. It can have only two values: 0 or 1.

A nibble is a group of 4 bits. A nibble can represent 16 different values, from 0000 to 1111.

A byte is a group of 8 bits. One byte can represent 256 different combinations.

Bit: 1 binary digit
Nibble: 4 bits
Byte: 8 bits

Applications of Bits

  • Boolean values: true/false, yes/no, on/off.
  • Logic circuits: flip-flops store 1 bit each.
  • Control signals: one bit can turn a device on or off.

Applications of Nibbles

  • Hexadecimal conversion: 4 binary bits equal 1 hex digit.
  • Compact representation: useful for reading and writing binary data easily.
  • Digital displays: often used in small groups for number coding.

Applications of Bytes

  • Character storage: one letter or symbol is usually stored in 1 byte.
  • Memory and storage: file sizes, RAM, and disk space are measured in bytes and their multiples.
  • Data transfer: network speed and communication data are often based on bytes or bits.

Examples

1 bit = 2 values
4 bits = 1 nibble = 16 values
8 bits = 1 byte = 256 values

Decimal to Binary Conversion

Divide by 2 repeatedly and record remainders. Read them from bottom to top.

Example: 13
13 ÷ 2 = 6 remainder 1
6 ÷ 2 = 3 remainder 0
3 ÷ 2 = 1 remainder 1
1 ÷ 2 = 0 remainder 1
Read upward → 13 in binary = 1101

Binary to Decimal Conversion

To convert binary into decimal, multiply each bit by its place value and add the results. The place values from right to left are 1, 2, 4, 8, 16, and so on.

Rule: Decimal = (bit × place value) added from right to left

Example 1: 1010

1010 = (1 × 8) + (0 × 4) + (1 × 2) + (0 × 1)
= 8 + 0 + 2 + 0
= 10

Example 2: 1101

1101 = (1 × 8) + (1 × 4) + (0 × 2) + (1 × 1)
= 8 + 4 + 0 + 1
= 13

Example 3: 10011

10011 = (1 × 16) + (0 × 8) + (0 × 4) + (1 × 2) + (1 × 1)
= 16 + 0 + 0 + 2 + 1
= 19

Hexadecimal Number System

The hexadecimal system uses 16 symbols: 0 to 9 and A to F. Here, A = 10, B = 11, C = 12, D = 13, E = 14, and F = 15. It is widely used in computers because one hex digit represents exactly 4 binary bits.

Why hex is useful: It makes long binary numbers shorter and easier to read.

Binary to Hexadecimal

To convert binary to hex, group the binary digits into sets of 4 from right to left, then convert each group to one hex digit.

Example 1: 1010

1010 = A

Example 2: 11010110

1101 0110 = D6

Example 3: 111100001010

1111 0000 1010 = F0A

Hexadecimal to Decimal

Each hex digit has a place value based on powers of 16.

Example 4: 2F

2F = (2 × 16) + (15 × 1)
= 32 + 15
= 47
