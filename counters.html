Counter Theory

A counter is a sequential circuit that goes through a prescribed sequence of binary states when triggered by a clock signal. Counters are used in timers, frequency dividers, address generators, and digital clocks.

Binary Ripple Counter

An asynchronous counter where each flip-flop is clocked by the output of the previous one.

Ring Counter

A shift register where the output of the last flip-flop is fed back to the first.

Only one flip-flop is HIGH at a time.

Johnson Counter

Also called a twisted ring counter. The complemented output of the last flip-flop is fed back to the first.

It gives 2N states for N flip-flops.

Up/Down Counter

Counts up or down based on mode input M.

UP: Q0(next) = Q0̄
Q1(next) = Q1 XOR Q0

DOWN: Q0(next) = Q0̄
Q1(next) = Q1 XOR Q0̄

MOD-N Counter

Counts from 0 to N-1 and then resets.

Number of flip-flops needed: ⌈log₂(N)⌉
